This repository contains a collection of **Exploratory Data Analysis (EDA)** notebooks created while learning and practicing data analysis using **Python and Pandas**.
Each notebook focuses on understanding datasets through structured exploration, asking relevant questions, and extracting meaningful insights from data.

---

## What This Repository Covers

- Data loading and inspection  
- Understanding dataset structure and features  
- Handling and exploring numerical and categorical data  
- Basic feature extraction from text-based columns  
- Summary statistics and exploratory questions  
- Introductory data visualization using Matplotlib and Seaborn
